# Private-Ticket-Bot
A simple ticket bot developed with discord.js v13

# How to install?
```
npm install
```
# How to config?
```
{
    "bot": {
        "token": "TOKEN",
        "clientid": "CLIENT ID"
    },

    "server": {
        "idguild": "SERVER ID GUILD"
    },

    "categorie": {
        "ticket": "TICKET CATEGORY ID"
    },
    
    "ruoli": {
        "staff": "STAFF ROLE ID"
    },

    "stanze": {
        "transcript": "TRANSCRIPT CHANNEL ID"
    }

}
  
```
# How to start?
```
node index.js
or
node .
```
